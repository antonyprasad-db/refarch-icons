# refarch-icons

Public-hosted icon set for Databricks-on-AWS reference-architecture slides
(Google Slides `createImage` needs a publicly-fetchable URL; org Drive DLP
blocks Drive public-sharing, so icons are served from raw.githubusercontent).

- `icons/aws-*.png` — official AWS Architecture Service icons (64px)
- `icons/dbx-*.png` — Databricks product icons (dark_cyan_3 variant)

Used by the PS QBR 2026 / GovCloud Better Together ref-arch slide.
